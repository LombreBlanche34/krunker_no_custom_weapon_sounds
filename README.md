# no_custom-weapon-sounds.js

Force the default sounds (default of the mod if there is one loaded) when there is a custom weapon sounds, no more quacker or any shits
You can choose for ennemi sounds and your sounds

### localstorage
`lombre_no_customWeaponSounds_status: true,`

`lombre_no_customWeaponSounds_my_weapons: true,`

`lombre_no_customWeaponSounds_ennemi_weapons: true`

### Compatible with [settings.js](https://github.com/LombreBlanche34/krunker_scripts_settings)
